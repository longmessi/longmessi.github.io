Serial Interface

  

BatteryMon is able to read information about a battery that is connected via the serial port. However due to the fact that there are subtle differences between the way each battery reports information through the serial port this feature will not work for most batteries. As such, this feature has been disabled for the public release.  If you are a developer the wants to use BatteryMon to read information from a specific battery type using the serial port than you can contact us for more information, see Contacting PassMark Software.
