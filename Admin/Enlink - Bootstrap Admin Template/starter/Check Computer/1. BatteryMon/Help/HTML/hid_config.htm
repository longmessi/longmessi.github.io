Configuration

    

Sample time: This setting determines how often BatteryMon should sample the battery information. If auto logging is turned on, this value will also determine how often the log file is updated.

Low battery % flash threshold: When the % Life left of the battery drops below this level the mini BatteryMon window will begin to flash.

Logfile Location: Specify where BatteryMon should save the log file to. The default location is "My Documents\PassMark\BatteryMon\batterymonlog.txt"

Run BatteryMon at Startup:  Automatically starts BatteryMon when Windows starts.

Auto logging: If the auto logging check box is checked, then this file will be updated in the background after each battery sampling

Start sampling when run: BatteryMon will automatically start taking samples when it is run, removing the need to click ‘start’, same as the command line parameter /r.

Start mini BatteryMon when run: BatteryMon will shrink to the mini monitoring mode on startup, same as the command line parameter /m.

Mini BatteryMon Always on Top: Having this option on ensures the mini BatteryMon window is always displayed on top of other windows.

Minimise to system tray only: When this option is checked and BatteryMon is minimised, the main window will no longer show up in the taskbar and only the icon will be displayed in the system tray.

Stop system from sleeping: Choosing this option will stop the computer from going into a sleep state while BatteryMon is running.

Show icon in system tray: Show an icon in the system tray when BatteryMon is running that changes depending on the current charge level and the charge status.

When auto logging, append date to filename: Appends the date to the log file for long term logging purposes.

Play sound when battery below certain level: Selecting this option will play a sound (once) when the selected battery drops below a certain level.

Level %: If the % Life left of the battery drops below this level a sound will be played.

Sound File: This is the sound to play.

Transparency: The amount of transparency determines how transparent the mini BatteryMon window is, 0% is solid and 100% is totally transparent.

 

Notification options: These options allow you to have the program alert you when certain criteria are met and to choose how you are alerted. The criteria available are,

•Battery percent below "x"%
•Voltage below "x"V
•Temperature greater than "x" °C
•Discharge Rate Above "x"mW

Where "x" is a configurable value. Note that Battery percent and discharge rate are based on aggregate data from all batteries whilst voltage and temperature are notified on a per battery basis. The discharge rate will not trigger until the battery has been discharging for at least two minutes due to possible fluctuations in estimates early in the discharge cycle, also, the discharge rate is not always measured in mW (see Charge/Discharge Units) When one of these conditions has been met there are several actions that can be taken,

•Do  Nothing: Disables the notification.
•Email: Sends an email detailing which criteria has been met and on which computer.
•Log: Writes a message to the log detailing the criteria met.
•Email+Log: Performs both Email and Log actions.
•Popup Alert: Generates a popup message box detailing the criteria met.

 

Edit email options: This button will open a dialog to changes the emails settings for notifications. These are;

Email List: This contains to list of addresses to send the email to when necessary. To add an email type the address in the field to the right of the list and click the “Add” button. To remove one or more names from the list, select them and click the “Delete” button.

         Outgoing mail server (SMTP): This is the mail server name for outgoing mail

         Port: Port number for the outgoing mail server

Sample Method: This is to choose how BatteryMon reads information from the battery. Internal is the default and recommended way where BatteryMon will query any batteries that are attached. Serial reads data from the serial port. Currently serial mode is not available, for more information see Serial Interface
