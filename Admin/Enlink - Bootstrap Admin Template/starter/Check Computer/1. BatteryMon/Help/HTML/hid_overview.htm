BatteryMon by PassMark Software - Overview

  

BatteryMon is an easy to use Windows based application that allows users to monitor the performance of battery systems being used to power portable computers and compatible uninterruptible power supplies connected to servers. The battery charge level is graphed in real time, along with an extrapolated trend line and comparison trend line. A log file is maintained of the battery charge level and other statistics for later analysis.
