Charge/Discharge Units

  

Where possible the charge and discharge rate is given in mW. However, due to the limitations of some batteries this is not always possible. In these cases the rates will be given in arbitrary units. In most cases BatteryMon will give an indication as to which unit is being used. If mW are being reported then mW will be printed after the value and if the units are arbitrary no unit type is given. The only time BatteryMon will not make a distinction between the two is in the log file records.
