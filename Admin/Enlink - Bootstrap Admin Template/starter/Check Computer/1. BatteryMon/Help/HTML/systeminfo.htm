System Information

  

Processor Type – The type of the detected processor.

Calculated Processor Speed – The processor speed as calculated by BatteryMon by performing direct calculations on the cpu. Please be aware that some portable computers are able to vary their speed depending on load so these figures may vary from time to time.

Reported Processor Max - Maximum specified clock frequency, in MHz, of the system processor as reported by the operating system.

Reported Processor Current - Processor clock frequency, in MHz as reported by the operating system. This number is the maximum specified processor clock frequency multiplied by the current processor throttle.

UPS present - Indicates the presence of an uninterruptible power supply (UPS).

Processor throttling - Indicates the system supports processor throttling.

Processor min throttle - Minimum level of system processor throttling supported, expressed as a percentage.

Processor max throttle - Maximum level of system processor throttling supported, expressed as a percentage.

Thermal zones - Indicates the system supports thermal zones. Thermal zones are areas in the system that contain temperature sensors to monitor the current temperature of those areas.

Video display dimming - Indicates the system supports video display dimming capabilities.

APM BIOS power management - Indicates the system supports APM BIOS power management features.

Removal of power to fixed disk devices - Indicates the system supports allowing the removal of power to fixed disk devices.

Fan throttle tolerance % - Lower limit, expressed as a percentage, that the processor may be throttled down to prior to turning on system fans in response to a thermal event.
